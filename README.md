# 电子宠物集合 🐕🥚

## 项目包含

### 1. 比格犬电子宠物
- **文件**: `beagle-pet.html`
- **功能**: 喂养、玩耍、摸摸、表演
- **特色**: 可爱的比格犬 SVG 图像，完全自包含

### 2. 拓麻歌子 Uni 复刻版
- **文件**: `tamagotchi-uni.html`  
- **功能**: 喂食、玩耍、清洁、睡觉、药品、状态检查
- **特色**: 经典拓麻歌子体验，完整的生命周期管理

## 部署说明
所有文件都已配置为 GitHub Pages 就绪，直接访问：
- 比格犬宠物: `https://bitchen6.github.io/beagle-pet.html`
- 拓麻歌子 Uni: `https://bitchen6.github.io/tamagotchi-uni.html`

## 技术特点
- 纯 HTML/CSS/JavaScript，无需外部依赖
- 响应式设计，支持手机和桌面
- 完整的游戏逻辑和状态管理
- 本地存储支持（可选扩展）
